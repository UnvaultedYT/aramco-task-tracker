Saudi Aramco Team Task Tracker 🏭
A beautiful, responsive task management dashboard built for Saudi Aramco teams to track and manage project tasks with real-time progress updates.

Features ✨
Team Dashboard: View team member profiles with individual task progress
Task Management: Track tasks by type (PM/CM) and status (Pending/In Progress/Done)
Admin Authentication: Secure admin access to close tasks with password protection
Real-time Statistics: Live progress tracking with percentage completion
Filtering & Sorting: Filter tasks by type and status for better organization
RTL Support: Full Arabic/RTL interface with modern UI
Responsive Design: Works seamlessly on desktop and mobile devices
Getting Started 🚀
Open index.html in your web browser
Browse through team members and task lists
Click "دخول المدير" (Admin Login) to unlock task closing functionality
Enter password: aramco2025
Close tasks as they are completed
